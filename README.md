# SecqurAIse

A simple event handler demo project.

[Visit website](https://secquraise-pj.web.app)

## FEATURES

- One can filter data
- Dark mode added for dark mode lovers
- Loading skeleton added to avoid layout shift
- Fully responsive

## TECHNOLOGY

- `react.js`
- `react-router`
- `material ui`
- `firebase`
- `react-use-size`
